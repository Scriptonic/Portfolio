Auto-Cropper

Through the use of Python and the package PyTorch, I created an automatic image cropper. The program works by taking in an image, locating the user's head using image recognition and a neural network. After the program finds the correct pixel location the photo gets cropped so that only the user's shoulders and head is visible. This is currently being used at the company I work for to standardize the profile pictures on their seating chart application.
